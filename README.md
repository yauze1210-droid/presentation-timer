## このツールについて

このツールは、プレゼン練習中に  
今の進行ペースが色で把握できるシンプルな補助ツールです。

原稿（プレゼンや発表用の台本）を読みながら時間を確認すると集中が切れやすいため、  
数字や細かい情報は表示せず、進行状況のみを色で示します。

[ツールはこちら](https://mamamakoto-guide.github.io/presentation-timer/)
---

## できること

- 台本を表示して練習を開始できます  
- 開始後は操作なしで進行します  
- 練習中のペースを色（🟢🟡🔴）で確認できます  

色は「今のペースが大きく崩れていないか」を  
感覚的に把握するための目安です。

---

## 特徴

- ストップウォッチや残り時間は表示しません  
- 練習中に追加操作は不要です
- 視界の端で進行状況が分かる設計です  

---

## 想定している使い方

- プレゼンや発表の個人練習  
- 時間配分に不安のある場合の読み練習  
- ペース感覚を掴むための補助用途  

正確な計測や記録を目的としたツールではありません。

---

## 注意点

- 表示される色は目安です  
- 読み進め方や話す速度によって、体感と差が出る場合があります  

進行状況の確認を補助する用途として利用してください。
